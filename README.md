#SnakeAdBlock
贪吃蛇大作战小米应用商店版（包名 `com.wepie.snake.new.mi`）去广告 Xposed 插件。

## Usage

Android 11+ 建议使用 [Lsposed](https://github.com/LSPosed/LSPosed)。
